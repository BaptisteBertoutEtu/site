Chasseur VS Monstre
Ce projet est un jeu tour à tour dans un labyrinthe. Le monstre doit sortir de celui-ci tandis que le chasseur fait tout pour le trouver. ( Ce projet est en cours, seulement le GitHub est accessible)

Lien vers le jeu

Présentation

Lors de ce projet, nous étions en groupe de 5. Nous avons diviser le travail à faire en différentes parties pour que chaque membre de l'équipe puisse faire quelque-chose qu'il sait faire.

Le but de ce projet etait de nous faire développer une application de jeu. Ce jeu est un jeu tour par tour entre deux personne, deux IA ou IA contre monstre/chasseur.

Nous avons réaliser une IHM grace à JavaFx. Par celui-ci le chasseur et le monstre peuvent choisir une case sur laquelle tirer/se déplacer.

Elle nous sert, en tant que joueur, à visualiser le labyrinthe et à pouvoir prendre nos décisions.

Le but pour le monstre est de parcourir le labyrinthe jusqu'à atteindre la sortie. Il a connaissance de la position de cette dernière.

Le chasseur quant à lui, doit tirer sur ce labyrinthe pour trouver la position du chasseur, s'il la trouve il a gagné le jeu.
